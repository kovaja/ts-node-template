# Kovaja template project
- node.js server with rest API, using express
- react frontend, with example of redux
- both in typescript

## Dev start up
- Start server
```
cd server
npm install
npm run dev
```
- Start client
```
cd client
npm install
npm run dev
```

## Deployment
- this template project will be deployed at heroku
